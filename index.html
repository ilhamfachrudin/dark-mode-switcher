DarkSwitch

Zero-dependency theme switcher

A lightweight TypeScript library for handling complex theme switching in web apps. Supports system preference detection, localStorage persistence, and smooth CSS transitions.

⚡ <1kb gzipped
🎨 CSS Variables
💾 Persistent
📱 System Aware
import { DarkSwitch } from 'dark-mode-switcher';

const ds = new DarkSwitch({ persist: true });
ds.toggle();  // Switch theme
ds.setTheme('dark');  // Force dark
ds.onThemeChange((t) => console.log(t));
